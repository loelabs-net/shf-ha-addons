# Smart Home Floorplan Addons for Home Assistant

This repository contains Home Assistant add-ons for Smart Home Floorplan, enabling seamless integration between your floorplan visualization and Home Assistant.

## Branches

- **main**: Stable - Recommended for daily use
- **edge**: Unstable - Latest features, may have bugs
- **dev**: Development - Experimental, likely broken

## Add-ons

### Smart Home Floorplan

The Smart Home Floorplan add-on provides a local proxy to Home Assistant.

## Installation

Add this repository to Home Assistant Supervisor:

1. Go to **Supervisor** → **Add-on Store**
2. Click the three dots (⋮) in the top right
3. Select **Repositories**
4. Add this repository URL: `https://github.com/loelabs-net/shf-ha-addons`
5. The Smart Home Floorplan add-on will appear in the add-on store
